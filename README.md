## Introduction

A Vue3 implement of the rap video of the author of Vue.js

## Install

```sh
# via npm
npm install --save v-rap

# via yarn
yarn add v-rap

# via pnpm
pnpm install v-rap
```

## Rap component

```vue
<script setup>
  import { Rap } from 'v-rap'
</script>

<VRap />
```
